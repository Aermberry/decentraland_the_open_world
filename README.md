## Decentraland 开发环境配置
---
### 安装Node.js(版本14或更高版本)
请前往[nodejs](https://nodejs.org/en/download/)官网下载安装。

安装完成后，请在终端输入:
```shell
    node -v
```
若能正常返回对应的版本，则已正确安装完成。
### 安装 CLI
在终端输入：
```shell
    npm install -g decentraland
```
全局安装Decentraland CLI。

检测安装版本：
```shell
    dcl -v
```
### 启动项目
```shell
    dcl start
```
